# ServerControl 使用说明

## 欢迎

ServerControl 是一款专为非技术用户设计的远程服务器管理工具。通过简洁的图形界面，您无需记忆任何命令，只需点击按钮即可完成常见的服务器管理操作。

---

## 安装方式

### 方式一：直接运行（需要 Python 环境）

1. 确保已安装 Python 3.10 或更高版本
2. 打开命令提示符，进入项目目录：
   ```
   cd ServerControl
   ```
3. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
4. 启动程序：
   ```
   python src/main.py
   ```

### 方式二：使用打包好的 exe 文件（推荐）

直接使用打包好的 `ServerControl.exe`，无需安装 Python，双击即可运行。
（打包方法见文末"打包为独立可执行文件"章节）

---

## 首次使用

程序首次启动时会自动弹出 **配置向导**，引导您完成以下步骤：

### 第 1 步：填写服务器信息

在向导中填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 连接名称 | 自定义标识（方便记忆） | 生产服务器 |
| 服务器地址 | IP 地址或域名 | 192.168.1.100 |
| SSH 端口 | 默认 22，如有变动请告知 | 22 |
| 用户名 | 登录用户名 | root |
| 密码 | 登录密码（或选择私钥文件） | ******** |

### 第 2 步：测试连接

点击 **"开始测试连接"** 按钮：
- ✅ 显示"连接成功" → 可以继续
- ❌ 显示错误信息 → 请检查填写的信息是否正确

### 第 3 步：完成配置

点击 **"完成"** 按钮，配置将自动加密保存。

### 连接服务器

1. 从顶部下拉框选择已保存的连接
2. 点击 **"🔗 连接"** 按钮
3. 等待状态栏显示 **"🟢 已连接"**

### 浏览远程目录

- 左侧目录树会显示服务器的文件夹结构
- 点击文件夹前的三角箭头可以展开子目录
- 点击某个目录后，点击 **"✓ 选择为工作目录"** 按钮

### 执行命令

- 右侧命令面板按分类显示所有预设命令
- 点击命令按钮即可执行
- 部分重要操作（如重启服务）会弹出确认对话框
- 执行结果实时显示在下方输出控制台

### 输出控制台

- 蓝色文字：系统消息（时间戳 + 操作说明）
- 白色文字：命令正常输出
- 红色文字：错误信息
- 点击 **"📋 复制"** 可复制输出内容
- 点击 **"🗑 清空"** 可清除所有输出

### 断开连接

点击顶部 **"⛓ 断开"** 按钮，或直接关闭窗口（程序会自动断开）。

## 自定义命令

您可以通过编辑配置文件来添加或修改命令。

### 方法

1. 点击菜单 **"工具" → "编辑命令配置"**，查看配置文件路径
2. 用记事本打开 `config/commands.json` 文件
3. 按格式添加或修改命令
4. 保存文件后重启程序生效

### 命令配置格式

```json
{
  "id": "my_command",
  "name": "按钮显示名称",
  "description": "命令说明（鼠标悬停显示）",
  "icon": "start",
  "command": "这里填写实际执行的命令",
  "category": "分类名称",
  "confirm": true,
  "confirm_message": "确定要执行吗？"
}
```

### 可用图标

| 图标名 | 显示 | 适用场景 |
|--------|------|---------|
| start | ▶ | 启动操作 |
| stop | ⏹ | 停止操作 |
| restart | 🔄 | 重启操作 |
| log | 📋 | 查看日志 |
| status | 📊 | 状态查询 |

## 安全说明

- 您的密码和连接信息会被 **加密存储** 在本机，不会发送到任何外部服务器
- 优先使用 Windows 凭据管理器存储密码（最安全）
- 如果凭据管理器不可用，会使用加密文件存储（位于 `%APPDATA%\ServerControl\`）
- 程序不会在任何地方记录或传输您的明文密码

## 常见问题

### Q: 连接超时怎么办？
A: 请检查：
1. 服务器 IP 地址是否正确
2. 网络是否通畅（可以尝试 ping 服务器）
3. 防火墙是否阻止了 SSH 端口
4. SSH 端口号是否正确

### Q: 密码错误？
A: 请确认：
1. 用户名和密码是否正确
2. 是否区分大小写
3. 如果使用密钥文件，请确认选择了正确的文件

### Q: 命令执行没有输出？
A: 可能是：
1. 命令本身没有输出（如启动/停止服务）
2. 命令执行时间较长，请耐心等待
3. 可以点击"取消命令"按钮终止

### Q: 如何添加新的服务器？
A: 点击菜单 **"文件" → "新建连接"**，填写新的服务器信息即可。

## 打包为独立可执行文件

如果您希望将程序打包为 exe 文件，让用户无需安装 Python 即可运行：

### 步骤

1. **安装 PyInstaller**：
   ```
   pip install pyinstaller
   ```

2. **执行打包**（或直接运行 `build.bat`）：
   ```
   pyinstaller --onefile --windowed --name "ServerControl" --icon "assets/app_icon.ico" --add-data "assets;assets" --add-data "config;config" --hidden-import "keyring.backends.windows" --hidden-import "paramiko.transport" --noconfirm src/main.py
   ```

3. **获取 exe 文件**：
   打包完成后，exe 文件位于 `dist/ServerControl.exe`

### 参数说明

| 参数 | 作用 |
|------|------|
| `--onefile` | 打包为单个 exe 文件 |
| `--windowed` | 不显示控制台窗口（关键！） |
| `--name` | 输出文件名 |
| `--icon` | 程序图标 |
| `--add-data` | 打包资源文件 |
| `--hidden-import` | 强制包含的模块 |

### 注意事项

- 打包后 exe 约 30-50 MB（PyQt6 体积较大）
- 首次运行可能较慢（解压临时文件）
- 配置文件（commands.json）需要在 exe 同级目录的 config 文件夹中
